# Космический Todo List - PRD

## Оригинальная задача
Создать красивый Todo List с анимациями на React + TypeScript + Vite. Адаптивная верстка, анимированный фон с размытыми сине-фиолетовыми пятнами, тёмная тема.

## Выбор пользователя
- Функции: базовые (добавление, удаление, отметка выполнения)
- Хранение: в памяти браузера (без persistence)
- Фон: анимированные размытые сине-фиолетовые пятна (blobs)
- Цветовая схема: сине-фиолетовая тёмная тема
- Иконки: SVG (без эмодзи)

## Архитектура
- **Frontend**: React 18 + TypeScript + Vite 6
- **Стили**: Tailwind CSS 4 с кастомными CSS переменными
- **Анимации**: Framer Motion + CSS keyframes для фона
- **Иконки**: Кастомные SVG компоненты с градиентами

## Реализованные функции (06.02.2026)
- Добавление задач через input + кнопку
- Отметка задач как выполненных (checkbox с анимацией)
- Удаление задач (hover для показа кнопки)
- Счётчики активных и выполненных задач
- Анимированный фон с размытыми сине-фиолетовыми пятнами
- Glassmorphism эффект на карточке
- Адаптивная верстка (mobile/desktop)
- Плавные анимации при всех действиях
- Кастомные SVG иконки с градиентами

## Бэклог
### P1 (высокий приоритет)
- Фильтрация задач (все/активные/выполненные)
- Редактирование текста задачи

### P2 (средний приоритет)
- LocalStorage persistence (опционально)
- Drag & drop для сортировки
- Категории/теги
